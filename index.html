What is this?

Better Place supports Templates up to 1024 by 1024 Pixels.

If your template is bigger it will be scaled down to fit this resolution.

This is usually not a problem, but for bigger pixel art projects this can mess the template up.

This Tool allows you to supply a large image and split it into seperate chunks, each being within the 1024x1024 template size limit.

Parameters Explanation:

Project Name: What your output files will be named.

Start Position: The absolute top left position all your chunks will reference in the template files.

Chunk Size: The size of each template.

How to use:

1. Select your image

2. Configure Parameters

3. Click the "Start" Button

4. Download individual templates by clicking on them or use the "Download all" Button
